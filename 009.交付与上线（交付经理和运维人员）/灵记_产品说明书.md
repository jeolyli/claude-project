# 灵记 (LingJi) 产品说明书

> 版本: V1.0 | 日期: 2026-06-16 | 类型: 个人财务管理工具

---

## 一、产品介绍

### 1.1 产品定位

**灵记 (LingJi)** 是一款采用"软萌粉嫩"设计语言的个人财务管理应用。以"治愈系财务"为品牌理念，帮助用户轻松记录日常收支、管理月度预算、分析消费习惯。

### 1.2 产品形态

灵记提供两种客户端形态，共享同一后端服务：

| 客户端 | 技术 | 覆盖平台 |
|--------|------|---------|
| **Web 浏览器版** | 原生 JavaScript MPA | Chrome / Safari / Firefox / 移动浏览器 |
| **跨端 APP 版** | uni-app (Vue 3) | iOS / Android / 微信小程序 |

### 1.3 用户角色

| 角色 | 描述 |
|------|------|
| **普通用户** | 注册登录、记录收支、管理预算、查看统计、管理分类 |
| **预设系统** | 提供 8 个支出大类 + 4 个收入大类 + 40+ 子分类作为初始数据 |

### 1.4 设计风格

- **色彩**: 樱花粉主色调 (#FF85A2)，搭配暖灰中性色
- **圆角**: 全界面无锐角 (8px–24px)，按钮胶囊形
- **字体**: Plus Jakarta Sans (标题) + Be Vietnam Pro (正文)
- **吉祥物**: 🐰 灵灵 — 在启动页、空状态、统计页出现

### 3.2 核心功能详解

#### 3.2.1 快速记账 (< 3 步)

```
步骤 1: 选择类型 (默认支出，可点击切换为收入)
步骤 2: 输入金额 + 选择分类 (点击分类图标)

Web 版: 保存后显示飘浮爱心/星星/花瓣动画
APP 版: 保存后弹出 Toast "💸 支出已记录"
```

#### 3.2.2 预算看板

```
进度条颜色阈值:
  🟢 绿色 (< 60%): 健康 👌
  🟡 黄色 (60-85%): 关注 👀
  🟠 橙色 (85-100%): 警告 ⚠️
  🔴 红色 (≥ 100%): 超支 🚨

看板展示:
  - 本月已支出: ¥X,XXX
  - 剩余可用: ¥X,XXX
  - 本月收入: ¥X,XXX
  - 进度条 + 百分比 + 健康度标签
```

#### 3.2.3 账单筛选

支持的筛选维度:
- **搜索**: 匹配金额、分类名、备注
- **类型**: 全部 / 支出 / 收入
- **时间**: 全部时间 / 今天 / 本周 / 本月 / 3个月 / 6个月 / 今年 / 自定义范围
- **分类**: 按具体分类筛选

结果按时间分组显示: 今天 → 昨天 → 本周 → 本月 → 更早

#### 3.2.4 数据同步机制

```
前端 localStorage (权威数据源)
        ↓ pushDataToBackend()
    Java 后端 MySQL (持久化)
        ↓ pullDataFromBackend()
前端 localStorage (恢复数据)

特性:
  - 同步是异步触发即忘 (fire-and-forget)
  - 失败静默降级，不阻塞用户操作
  - 通过 reference_id 去重，防止重复导入
```

## 四、用户操作说明

### 4.1 Web 浏览器版

#### 启动方式

```
方式一 (推荐): Express 后端提供静态服务
  cd 003.前端代码（前端工程师）/backend
  npm run dev
  浏览器打开: http://localhost:3001

方式二: 直接打开文件
  浏览器打开: frontend/pages/login.html
```

#### 首次使用

1. 打开登录页 → 点击"立即注册"
2. 输入用户名 (2-20 字符) 和密码 (8 位以上，含字母和数字)
3. 注册成功后自动跳转首页
4. 首页显示空预算 (默认 ¥5,000) 和预设分类

#### 记一笔账

1. 首页"快速记账"区域，默认选中"支出"
2. 点击金额输入框，输入数字
3. 在下方分类栏点击分类图标 (如 🍽️ 餐饮)
4. 可选：输入备注
5. 点击"💾 保存记录"

#### 设置预算

1. 首页 → 底部"我的" → "预算设置"
2. 修改月度总预算
3. 为各分类设置预算上限
4. 查看分类合计与差额
5. 点击"💾 保存预算"

#### 查看统计

1. 底部导航 → "统计"
2. 切换"本周"/"本月"查看不同时间范围
3. 查看收支汇总和图表

#### 导出数据

1. 底部导航 → "我的" → "数据管理"
2. "导出 CSV" — 生成带中文支持的 CSV 文件
3. "备份数据 (JSON)" — 下载完整数据备份
4. "从备份恢复" — 选择 JSON 文件恢复

### 4.2 跨端 APP 版

#### 启动方式

```
1. 用 HBuilderX 打开 uniapp-lingji 目录
2. 运行 → 运行到手机或模拟器 → Android App / iOS App
3. 或 运行 → 运行到小程序模拟器 → 微信开发者工具
```

#### Android 真机调试

```
1. 手机与电脑连接同一 WiFi
2. 修改 utils/api.js 中的 HOST_IP 为电脑局域网 IP
3. HBuilderX 运行到 Android 真机
```

#### 操作说明

APP 版与 Web 版功能完全一致，操作方式相同。主要差异:
- 登录页无浮动装饰背景 (Android 兼容性)
- 统计图表使用 SVG (非 Chart.js)
- 数据备份使用系统文件管理器 (非浏览器下载)
- 头像上传调用系统相机/相册

## 五、常见问题

### 5.1 登录相关

**Q: 登录时提示"网络错误"?**
A: 检查 Java 后端是否启动 (`http://localhost:8086`)。确认 `005.后端代码（Java工程师）` 已通过 `mvn spring-boot:run` 启动。

**Q: 登录失败超过 5 次被锁定?**
A: 锁定时间为 15 分钟。可以通过清除浏览器 localStorage 中的 `lingji_login_fail` 键立即解除。

**Q: 密码忘记了怎么办?**
A: 当前版本不支持密码找回。请联系管理员重置，或使用新账号重新注册。

### 5.2 数据相关

**Q: 我的数据存在哪里?**
A: 
- Web 版: 数据存储在浏览器 localStorage 中，同时异步同步到 Java 后端 MySQL 数据库
- APP 版: 数据存储在手机本地存储中 (`uni.setStorageSync`)，同时异步同步到后端

**Q: 清除浏览器缓存会丢失数据吗?**
A: 会。localStorage 中的数据会被清除。建议定期使用"数据管理"→"备份数据(JSON)"导出备份。

**Q: 如何在新设备上恢复数据?**
A: 
1. 在旧设备上: 数据管理 → 备份数据(JSON) → 保存文件
2. 在新设备上: 登录同一账号 → 数据管理 → 从备份恢复 → 选择备份文件
3. 或者: 登录后自动从后端拉取已同步的数据

**Q: CSV 导出中文乱码?**
A: 导出的 CSV 包含 UTF-8 BOM 头，用 Excel 直接打开即可正常显示。如仍有乱码，用记事本打开后另存为 UTF-8 编码。

### 5.3 APP 版专有问题

**Q: Android 上输入框点不动?**
A: 已修复。如仍遇到，尝试点击输入框所在行 (而非仅点击输入框内部) 来触发键盘。

**Q: 微信小程序图表不显示?**
A: 微信小程序不支持 SVG data URI 格式的图表。该平台暂不支持图表功能，可在 APP 或 H5 端查看。

**Q: Android 真机无法连接后端?**
A: 
1. 确认手机和电脑在同一 WiFi
2. 修改 `utils/api.js` 中 `HOST_IP` 为电脑 IP (如 `192.168.0.199`)
3. 确认电脑防火墙允许 8086 端口入站

**Q: 保存按钮灰色无法点击?**
A: 需要同时满足两个条件: (1) 输入金额 > 0，(2) 选择一个分类。满足后按钮自动变亮。

### 5.4 后端相关

**Q: 如何初始化数据库?**
A: 
```bash
mysql -u root -p < 005.后端代码（Java工程师）/src/main/resources/db/init.sql
```
这会创建 `lingji` 数据库和所有表，并插入预设分类种子数据。

**Q: 数据库连接失败?**
A: 检查 `application.yml` 中的数据库配置:
- 默认端口: 8806 (非标准 3306)
- 默认用户/密码: `root` / `root_123456`
- 数据库名: `lingji`

**Q: API 需要认证吗?**
A: 当前版本 (V1.0) 的 API 使用 `userId` 请求参数标识用户，未强制 JWT 令牌验证。后续版本将加入认证拦截器。

### 5.5 预算相关

**Q: 设置了分类预算但看板不显示?**
A: 确认:
1. 已保存预算 (点击"💾 保存预算")
2. 查看的月份与设置的月份一致 (默认当月)
3. 浏览器控制台是否有 `[sync] 已推送到后端` 日志

**Q: 同一月份多次设置预算会怎样?**
A: 新的预算会完全替换旧的预算 (先删除后插入)，不会产生重复记录。
