# Design Specification - Finance Manager Pitch Deck

## Project Info
- **Project**: finance-pitch
- **Format**: 16:9 (1280×720)
- **Language**: 中文

## Design Parameters

### Color Palette - Midnight Executive + Cyan Tech
| Role | Color | Usage |
|------|-------|-------|
| Primary | #1E3A5F | 主色调 - 深海军蓝 |
| Secondary | #00D4FF | 强调色 - 科技青 |
| Accent | #FF6B35 | 辅助强调 - 橙色 |
| Text Primary | #FFFFFF | 标题文字 |
| Text Secondary | #B8C5D6 | 正文文字 |
| Background | #0A1428 | 幻灯片背景 |
| Card BG | #152538 | 卡片背景 |
| Success | #00FF88 | 正向指标 |

### Typography
- **Title Font**: "Noto Sans SC", "Microsoft YaHei", sans-serif
- **Body Font**: "Noto Sans SC", sans-serif
- **H1 Size**: 48-56px (slide titles)
- **H2 Size**: 32-36px (section headers)
- **Body Size**: 16-18px
- **Caption Size**: 12-14px

### Layout
- Canvas: 1280×720
- Margins: 60px
- Content area: 1160×600
- Element spacing: 24px

### Visual Effects
- Cards: Rounded rectangles with subtle border glow
- Icons: Simple line icons
- Decorations: Gradient accent bars at top
